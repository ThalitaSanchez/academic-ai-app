# Contributing to Academic AI App

Obrigado por considerar contribuir para o Academic AI App!

## Como Contribuir

### Reportar Bugs
Crie uma issue com descrição clara do problema, passos para reproduzir e seu ambiente.

### Sugerir Melhorias
Sugestões são bem-vindas! Crie uma issue descrevendo a melhoria proposta.

### Pull Requests
1. Fork o repositório
2. Crie uma branch (`git checkout -b feature/AmazingFeature`)
3. Commit suas mudanças (`git commit -m 'Add some AmazingFeature'`)
4. Push para a branch (`git push origin feature/AmazingFeature`)
5. Abra um Pull Request

## Convenções

### Commit Messages
- `feat:` novas features
- `fix:` correções de bugs
- `docs:` documentação
- `test:` testes
- `refactor:` refatoração

### Branch Naming
- `feature/description`
- `bugfix/description`
- `docs/description`
